# **L1: E-Commerce Sales Analytics API**  

### **Objective:**  
Develop an API to analyze **sales trends and product performance** using MongoDB aggregations on a **single `orders` collection**.  

---

### **Schema Details:**  

#### **`orders` Collection**  
```js
const orderSchema = new mongoose.Schema({
  username: String,
  productName: String,
  category: String,
  quantity: Number,
  totalPrice: Number,
  orderDate: Date,
  status: String, // "Pending", "Shipped", "Delivered", "Cancelled"
});
```

---

### **Routes & Queries:**  

#### **1Ô∏è Data Insertion Route**  
- **POST `/orders`** ‚Üí Add a new order  

#### **2Ô∏è Aggregation Routes**  

- **GET `/analytics/top-products`** ‚Üí Find the **top 3 best-selling products** based on total quantity sold.  
- **GET `/analytics/revenue-by-category`** ‚Üí Compute **total revenue for each product category**.  
- **GET `/analytics/average-order-value`** ‚Üí Calculate **average total price of an order**.  
- **GET `/analytics/orders-per-month`** ‚Üí Count **number of orders placed each month**.  
- **GET `/analytics/cancellation-rate`** ‚Üí Find the **percentage of canceled orders**.
