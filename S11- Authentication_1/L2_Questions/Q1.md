# **L2: Role-Based Access Control - Todo App API**  

### **Objective:**  
Develop an API that implements **Role-Based Access Control (RBAC)** in a **Todo Management System**, where:  
- **Users** can create, read, update, and delete **only their own todos**.  
- **Admins** can perform **CRUD operations on all todos**.  

---

### **Schema Details:**

#### **`users` Collection**

```js
const userSchema = new mongoose.Schema({
  name: String,
  email: { type: String, unique: true },
  password: String,
  role: { type: String, enum: ["User", "Admin"], default: "User" }, // Role-based access
});
```

#### **`todos` Collection**

```js
const todoSchema = new mongoose.Schema({
  title: String,
  description: String,
  completed: { type: Boolean, default: false },
  createdBy: { type: mongoose.Schema.Types.ObjectId, ref: "User" },
});
```

---

### **Routes & Access Control:**

#### **1Ô∏è Authentication Routes**

- **POST `/signup`** ‚Üí Register a new user (default role: `"User"`)
- **POST `/login`** ‚Üí Authenticate user, generate a **JWT token**

#### **2Ô∏è Middleware for Role-Based Access**

- **Auth Middleware (`authMiddleware.js`)** ‚Üí Verifies JWT, attaches user info to `req.user`.
- **Admin Middleware (`adminMiddleware.js`)** ‚Üí Checks if `req.user.role === "Admin"` before proceeding.

#### **3Ô∏è Todo Routes (Protected by JWT)**

üîπ **User-Specific Actions**

- **POST `/todos`** ‚Üí **Create** a new todo (only for self).
- **GET `/todos`** ‚Üí **Fetch only the todos created by the logged-in user**.
- **PUT `/todos/:id`** ‚Üí **Update only self-created todos**.
- **DELETE `/todos/:id`** ‚Üí **Delete only self-created todos**.

üîπ **Admin-Only Actions**

- **GET `/admin/todos`** ‚Üí **Fetch all todos in the database** (Admin only).
- **DELETE `/admin/todos/:id`** ‚Üí **Delete any todo** (Admin only).

---

### **Expected Outcomes:**

- **Users can only manage their own todos.**
- **Admins can manage all todos in the system.**
- **Unauthorized users cannot access protected routes.**
- **JWT authentication is required for all routes.**
